# NagiosPlugins
Nagios Plugins for Nagios Core and Nagios XI


## check_last_nls_backup.sh

| Backup Status Nagios Log Server   | Status in Nagios XI (Last NLS Backup) |
|-----------------------------------|---------------------------------------|
| SUCCESS                           | OK                                    |
| PARTIAL                           | WARNING                               |
| FAILED                            | CRITICAL                              |
| IN_PROGRESS, NOT FOUND or UNKNOWN | UNKOWN                                |
